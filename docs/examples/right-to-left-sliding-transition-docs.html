Right to left sliding transition Guide

Example of a transition that slides from right to left instead of the default animation.

View this example

<!doctype html> <html> <head> <title>An example with
                                images sliding from right to left</title> <style> #myslider { width:612px;
                                height:612px; } </style> </head> <body> <div
                                id="myslider"> <img
                                src="http://placekitten.com/g/612/612"/> <img
                                src="https://unsplash.it/612/612"/> <img
                                src="https://unsplash.it/612/612?random=1"/> <img
                                src="https://unsplash.it/612/612?random=2"/> </div> </body>
                                <script src="../dist/simpleslider.js"></script> <script> var
                                imgSlider = simpleslider.getSlider({ container:
                                document.getElementById('myslider'), prop: 'left', init: 612, show: 0,
                                end: -612, unit: 'px', delay: 2, duration: 1 }); </script>
                                </html>
